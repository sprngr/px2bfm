Pixels 2 BitFontMaker2 (aka px2bfm) is a simple tool allowing you to use your desired pixel editor to draw font glyphs, creating compatible BitFontMaker2 JSON, and allowing you to import it to generate a new font.

Template Downloads: .ase (Aseprite) format | .png format

For more information as to how to use this, check the manual.

For more information about the project as a whole or the px2bfm npm package, check the GitHub repository.

Font Configuration Options

You may set some configuration options now or after it's imported later in BitFontMaker2.

Upload Sprite Sheet

Drag & drop an image or click the plus sign (+) to upload your sprite sheet of 16x16 font glyphs.

Copy JSON

Click the text below to have the generated JSON copied to your clipboard, and then import it into BitFontMaker2
